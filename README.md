# Brick Blast

키보드와 터치로 즐기는 클래식 벽돌 깨기 게임입니다.

## 주요 기능

- 빠른 공 이동에도 벽돌을 통과하지 않도록 보정한 충돌 처리
- 키보드 입력을 부드럽게 반영하는 패들 움직임
- 밝은/어두운 테마 전환
- 브라우저에 저장되는 로컬 랭킹

## 실행하기

```bash
npm install
npm run dev
```

## 검증 및 빌드

```bash
npm test
npm run build
```
